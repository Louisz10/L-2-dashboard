Import Square Transactions (CSV)
How to export your Square transactions:
1 Go to squareup.com/dashboard/sales/transactions and sign in
2 Click Sales → set your date range → click Export → Transactions CSV
3 Click Choose CSV File above — duplicates are automatically skipped

Import from Square Item Library
How to export your Square item library:
1 Go to squareup.com/dashboard/items/library
2 Click Export Library (top right) → Export as CSV
3 Click Choose CSV File above — existing items are updated, new ones are added
